# Event-Driven Cloud Data Processing Pipeline

This project demonstrates an event-driven data processing pipeline using AWS serverless services.
It includes architecture design, research documentation, infrastructure as code, CI/CD configuration,
and a walkthrough video.

## Key Concepts
- Event-driven architecture
- Serverless computing
- Infrastructure as Code (Terraform)
- CI/CD using GitHub Actions

This project was created as part of a cloud internship assignment.
